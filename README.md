# doit

## Setup

```.bashrc
export OPENAI_API_KEY=...
export CLOUDFLARE_AI_GATEWAY=
```

## Usage

Use `doit` Command:

```bash
doit echo hoge
```

## Examples

```bash
doit show files
```
